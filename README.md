# ResQNet – Disaster Management System

ResQNet is a full-stack web application designed to assist in disaster management by connecting users with shelters, resources, and emergency support.

## Features
- User authentication
- Shelter listing and search
- Disaster-related resource management
- Real-time data handling
- Interactive frontend UI

## Tech Stack
### Frontend
- React
- Vite
- Bootstrap
- Axios

### Backend
- Node.js
- Express.js
- MongoDB

## Project Structure
- resqnet-client → Frontend
- resqnet-api → Backend

cd resqnet-api
